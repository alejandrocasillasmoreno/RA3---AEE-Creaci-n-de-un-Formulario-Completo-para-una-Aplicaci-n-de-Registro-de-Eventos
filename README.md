#Descripción del Proyecto
Esta actividad consiste en crear un formulario web completo para el registro de asistentes a eventos, desarrollado como parte de la actividad evaluable RA3 del módulo DWEC.

##Características del Formulario
###6 secciones organizadas:

Información personal

Información del evento

Información de acceso

Preferencias de contacto

Encuesta adicional

Adjuntar un archivo

Diseño responsivo con Bootstrap

Validación integrada con atributos HTML5 required

Procesamiento de datos con PHP

##Tecnologías Utilizadas
###Lenguajes: HTML5, PHP

###Framework: Bootstrap para el diseño

###Entorno de pruebas: XAMPP

###Validación: Atributos HTML5 (required, pattern, type)

##Campos del Formulario
###1. Información Personal
Nombre completo (requerido)

Correo electrónico (requerido)

Número de teléfono

Fecha de nacimiento (requerida)

Género

###2. Información del Evento
Fecha del evento (requerida)

Tipo de entrada (General, VIP, Estudiante)

Preferencias de comida (Vegetariano, Vegano, Sin gluten, Sin preferencias)

###3. Información de Acceso
Nombre de usuario (requerido)

Contraseña (requerida)

Confirmación de contraseña (requerida)

###4. Preferencias de Contacto
Notificaciones por correo electrónico

Términos y condiciones (requerido)

###5. Encuesta Adicional
Calificación de eventos anteriores (rango 1-10)

Comentarios adicionales

###6. Adjuntar Archivo
Subir archivo (opcional)
